# Alpha Plaza Hotel - Landing Page

## Project Overview
High-end luxury landing page for Alpha Plaza Hotel with Obsidian & Gold aesthetic, focused on conversion and elite hospitality.

## Original Problem Statement
Create a luxury landing page for "Alpha Plaza Hotel" with:
- **Theme**: Dark Mode (Obsidian Black) with Alpha Gold (#D4AF37) accents
- **Aesthetic**: Minimalist, futuristic, sophisticated with glassmorphism effects
- **Target**: Conversion-optimized, $20,000+ agency-quality website
- **Location**: Taguatinga-DF, Brasília
- **Language**: Portuguese (BR)

## User Information

- **Address**: Avenida Central, Lote 1040, Núcleo Bandeirante, Brasília/DF

## Architecture & Tech Stack
- **Frontend**: React 19 with Tailwind CSS
- **Components**: Shadcn UI (Button, Card)
- **Styling**: Custom Obsidian & Gold theme with glassmorphism
- **Images**: Professional luxury hotel images from Unsplash
- **Deployment**: Frontend only (static landing page)
